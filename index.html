About this site

This portfolio site was designed in two hours as an experiment with flexbox layouts and SVG. Learn more about the techniques I use in the Portfolio section.

About me

I have managed web content for the 16th largest city in the United States, Fort Worth, Texas, since 2006, and have updated the front-end code to evolve with changing web practices.

I changed the site from a table-based layout to a CSS layout in 2008, and moved the city to a responsive version of the site in 2013.

While working as a one-person web shop responsible for more than 90 percent of website updates, I completed a full redesign from the ground up in 2015. The new design includes a flexbox layout and an SVG sprite for icons and core graphics. The site also utilizes the SASS CSS pre-processor and the PostCSS Autoprefixer plugin to ensure backward compatibility. All pages managed in my office are built using Jekyll, a static-site generator, and are version-controlled with Github.
